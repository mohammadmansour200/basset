## Features 🌟

- ✂️ **Media cutting and trimming with preview**
- 🔄 **Media conversion**
- 📉 **Media compression**
- 📉 **Video quality downgrading**
- ⚙️ **Full customizability**
- 🌐 **Multi-lingual support**
- 🌗 **Dark/Light mode**
- 🧠 **Low RAM usage (thanks to Tuari)**
- 🔒 **Privacy-focused (everything happens locally)**
